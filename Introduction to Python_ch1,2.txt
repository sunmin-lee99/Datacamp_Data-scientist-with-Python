[Introduction to Python]

1. Python Basics

<Python>
: 파이썬은 오픈소스(무료)로 어디에서나 사용하기 쉽기 때문에 널리 퍼졌다
: 계산을 쉽게 하므로 아주 좋은 계산기가 될 수 있다
: comment를 달때는 '#' 이용하면 인식하지 않는다


<Python as a calculator>
: 수학하듯이 문제에 해당하는 수식을 쓰고 수학기호 대신 그에 알맞은 파이썬 연산자를 넣으면 쉽게 계산할 수 있게 된다.

+ : Addition
- : Subtraction
* : Multiplication
/ : Division
% : Modulo (나머지)
** : Exponentiation (지수)


<Variable, 변수>
: 대소문자를 구분하는 특정 '이름'
: 더 복잡한 계산을 하기위해선 코딩하는 동안 값을 '저장' 해야함
=> 이 '저장'을 하기 위해 변수를 정의하여 그 안에 값을 '저장' 해야함
# 역할 : 이런 변수를 생성한 뒤 값을 지정해서 '저장'해두면, 이 변수를 입력할때마다 해당 값이 호출됨


<Type, 유형>
: 입력한 값의 유형을 알아보는 것
# 실행방법 : type()
# 출력하려면 꼭 앞에 'print' 붙이기

- float : 정수부분과 소수부분을 모두 가질 수 있는 숫자 (실수형)
- int : 정수형
- str : 문자열 ('',"" 모두 사용 가능)
- bool : True 또는 False를 나타내는 유형 (데이터에 대한 필터링 작업을 수행하는데 매우 유용)
# 주의해야할 것 1 : ()안에 문자가 쓰여있더라도 그 변수가 가지고 있는 값이 유형으로 나옴
ex. 
bmi = 3.23 
type(bmi) # bmi 자체는 문자지만 bmi에 저장되어있는 값이 실수이므로 float

float
# 주의해야할 것 2 : Different type = Different behavior!
(코드가 작업중인 유형에 따라 코드가 작동하는 방식이 다름)
ex.
2 + 3

5

'ab' + 'cd' # 문자열은 꼭 따옴표를 붙여줘야함

'abcd' 
# 결과값도 따옴표가 붙여진채 나옴
# 문자열의 덧셈 = 두 문자열이 순서대로 붙어서 나옴

-> 이렇게 정수형끼리 계산하면 정수형이 문자열끼리 더하면 문자열이 결과값으로 도출됨
(단, 문자열과 숫자의 덧셈을 할 경우는 에러가 발생함)

# 주의해야할 것 3 : 문자열과 숫자열끼리의 변환 (서로 덧셈 기호를 공유할때)
문자열로 변환 : str()
정수형으로 변환 : int()
실수형으로 변환 : float()
bool형으로 변환 : bool()

(숫자열을 문자열로 변환할때 괄호 안에 수식이 있더라도 모두 처리된 후 문자열로 변환됨)
-> 이렇게 어떤 type으로든 변환 가능
-> 그러나 덧셈에서 붙여주기, 곱셈에서 여러번 반복해서 쓰기만 가능할 뿐, 문자열을 숫자열로 바꾼다고 해서 계산되는 것은 아님

# True + False 
1 (결과값)

# 문자열 곱셈도 가능
ex.
"Hey" * 2

# 변수가 정해져있다면 정해진 부분은 건들지 말것
-> 바꾸더라도 print()이 부분을 바꿔줘야함
ex. 정수형을 문자열로 바꾸기
x = 5
